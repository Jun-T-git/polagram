# @polagram/preview

The preview interface for Polagram, built with Vite and React.
This package provides the web UI used by the `polagram preview` command.

## Features

- **Real-time Preview**: Renders Mermaid diagrams dynamically.
- **Interactive**: Supports zooming and panning.
- **Design**: Clean and modern UI for viewing diagrams.

## Development

Run locally:

```bash
pnpm dev
```

Build:

```bash
pnpm build
```
