# @polagram/preview

![npm version](https://img.shields.io/npm/v/@polagram/preview)

Internal static spec viewer for [Polagram](https://polagram.org).

## Disclaimer

This package is intended to be used internally by the `@polagram/cli` tool.
It is published to npm to resolve dependencies for consumers of the CLI, but it is **not designed for direct standalone usage**.

## Features

- Renders Polagram diagrams (Mermaid, etc.)
- Provides interactive preview controls
